## ⚡ Quick Start

Make sure Go is installed:

```bash
go version
# go version go1.21+ linux/amd64
```

Clone this repo:

```bash
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>
```

Run any example file:

```bash
go run examples/helloWorld.go
```

## 🧠 Why Go?

- Statically typed, compiled — fast and predictable
- Built-in concurrency via goroutines and channels
- Minimal syntax — easy to read, hard to misread
- First-class tooling: `go fmt`, `go test`, `go build` out of the box
- Widely used in cloud-native and backend systems (Docker, Kubernetes, krkn — all Go)
